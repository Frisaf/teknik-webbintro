HTML

HTML står för HyperText Markup Language och är det språk jag använt mig under detta projekt.
HTML består av så kallade taggar. Dessa beskriver hur varje sak ska se ut och hur den ska positioneras. Några exempel på taggar jag använt mig av är:

div

En div kan beskrivas som en "box", något man brukar prata mycket om när det gäller just HTML. Genom att dela upp sin sida i divar så kan man få en bättre fördelning som är enklare att jobba med när man ska få saker på specifika platser på sidan, t.ex min centrerade text och dess bakgrund. Om du vill titta mer på det rekommenderar jag att se på koden på GitHub

nav

Detta är en annan tagg som indikerar att innehållet innuti denna tagg är en navigationsbar, det vill säga en liten lista på vad som finns på sidan med hyperlänkar till dessa.

strong

Denna tagg indikerar att texten innuti den ska vara i fetstil.

em

Denna tagg liknar strong-taggen, men skillnaden är att denna indikerar att texten ska vara i kursiv stil. Förkortningen em står för emphasis, vilket är engelska för betoning.

Git / GitHub

Git är ett versionshanteringsverktyg som gör att man kan synka alla ändringar med medarbetare om man är flera som jobbar på ett projekt. I detta projekt har jag använt Git för att kunna ladda upp mitt projekt till GitHub, det vill säga en webbsida för git repositories.

Om du vill veta mer om vilka taggar jag använt, ta en titt på GitHub repositoryt och läs filen logging.md där jag förklarar mer om vilka git kommandon jag använt.

CSS

CSS är en förkortning och står för Cascading Style Sheets. CSS används för att skapa stilar till olika sorters markup språk, till exempel HTML som jag använt i detta projekt. Med CSS kan man skapa olika så kallade klasser som bestämmer hur exempelvis alla divar med denna tagg ska se ut. Man kan också med hjälp av CSS bestämma hur alla taggar av en specifik sort ska se ut. En klass har en punkt framför sig, sedan måsvingar och inuti måsvingarna olika regler för den klassen. Om man vill bestämma en stil för alla taggar av en sort tar man bara bort punkten. Resten är samma.

JavaScript

JavaScript är ett programmeringsspråk som ofta används till webben tillsammans med HTML och CSS. JavaScript används ofta för att få vissa funktioner på hemsidan att fungera, som t.ex min knapp som ändrar bakgrund på sidan.
